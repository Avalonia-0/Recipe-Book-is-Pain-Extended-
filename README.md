# Recipe-Book-is-Pain-Extended-
[Recipe Book is Pain](https://modrinth.com/mod/rbip)的高版本非官方分支

# 拓展功能
1. 新增配方书上侧和下侧的标签页
![示例](https://cdn.modrinth.com/data/cached_images/4adb7596ccd3bd1d719729be078cd7494dd6e6dc.png)  
2. 使用鼠标滚轮翻页
3. 搜索页常驻

## 配置`config\recipe-book-is-pain-extended.json`
1. 将"extended_features"改为false来禁用拓展功能
2. 修改"bottom_number"的值来改变显示的标签页的数量，最大16（如同图片展示的那样），最小6（原版）。
